Classic Film Archive Website

This website was created as a personal design and web UI/UX project, using copyright-free images, films from classic films whose rights have expired.

All film images and informations are sourced from IMDB.

Used muted green tones(#90aa8b), light-gray tones(#aaa, #bbb) to evoke simple and vintage mood.

Mixed "Modern Sans" with "SUIT" fonts to balance classic and modern aesthetics.

Website main section

  • Now - A Visual highlight of selected films
  • Best Now - Layout exploration with curated picks
  • Recommendation - Grid-based design with hover effects
  • Review - User review UI styling
  • Genre - List styling with interactive feedback
  • Articles - Horizontal slider with navigation arrows
